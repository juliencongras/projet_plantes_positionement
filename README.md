# Mes Plantes — Aperçu


Page d'accueil basique pour gérer des plantes localement (PHP + CSS).

Prévisualisation

Ouvrez `index.php` dans votre navigateur ou servez le dossier :

```bash
# depuis le répertoire du projet
php -S 127.0.0.1:8002 -t .
# puis ouvrez http://127.0.0.1:8002/index.php
```

Fichiers principaux:

- [index.php](index.php)
- [assets/css/styles.css](assets/css/styles.css)
